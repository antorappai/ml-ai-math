# ML Math Studio

React prototype for:

> Understand ML math without getting lost in formulas.

## Audience

Learners preparing for ML who may not have a strong school-math background.

## Included

- Vite + React app
- Reusable lesson structure based on the 11-part teaching format
- Interactive lessons for vectors, dot product, and projection
- Beginner-friendly symbol decoder in every lesson

## Run

```bash
npm install
npm run dev
```

Then open the local Vite URL shown in the terminal.

## Build

```bash
npm run build
```

## GitHub Pages
